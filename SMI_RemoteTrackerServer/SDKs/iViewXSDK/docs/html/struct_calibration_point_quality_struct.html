Detailed Description

This struct provides information about the fixation quality when a calibration point was shown.

If the request calibration quality data is not available, the number and positionX/positionY will be set to -1. User have to check these fields to make sure the returned data is valid.

Use iV_GetCalibrationQuality to retrieve the calibration quality data
